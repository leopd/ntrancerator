# N-Trancerator

A DJ Visualization tool.

## Prerequisites

- [Rust](https://rustup.rs/) (stable toolchain — installs via `rustup default stable`)

## Run

```sh
cargo run
```

## Build a release binary

```sh
cargo build --release
# binary at: target/release/ntrancerator
```

## Test

```sh
cargo test
```

## Lint & format

```sh
cargo clippy
cargo fmt
```
